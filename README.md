# next-multiterminal

基于[`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app)脚手架搭建的Next.js应用

## 启动步骤

安装依赖:

```bash
npm install pnpm -g
pnpm i
```

启动本地应用
```bash
pnpm dev
```
浏览器中打开 [http://localhost:3000](http://localhost:3000) 预览页面.

## 构建SSR版本
```bash
pnpm build
```

## 构建SSG版本
```bash
pnpm build-ssg
```
